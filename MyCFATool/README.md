# MyCFATool

## Overview

MyCFATool is a personal, institutional-grade financial data platform designed to automate the collection, storage, verification, analysis, and visualization of equity financial data.

## Features

- Automated data retrieval from Financial Modeling Prep (FMP)
- Storage in CSV and SQLite databases
- Python Dash dashboard for visualization
- Support for quarterly and annual data
- Financial ratios computation and validation
- Technical analysis and forecasting

## Project Structure

- `data/`: Raw and processed data
- `database/`: SQLite databases
- `ingestion/`: Data ingestion modules
- `analytics/`: Analysis and computation modules
- `dashboard/`: Dash application
- `config/`: Configuration files
- `tests/`: Unit tests
